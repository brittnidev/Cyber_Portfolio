Each project showcases hands-on work, problem-solving, and demonstrate my drive to learn and adapt to real-world tech environments.

What you’ll find in this repo
- Focused projects that cover foundational to intermediate IT topics
- Realistic tasks, configurations, scripts, or simulations that reflect industry practices
- Brief writeups that explain goals, approaches, outcomes, and learning reflections
- Artifacts (where applicable): sample configurations, test results, diagrams, and code snippets

- They emphasize my hands-on, results-driven mindset—vital for security roles

How to read the project entries
- Overview: Quick summary of the project’s objective and context
- What I did: Key steps, tools, and methods I used
- Artifacts: Links or references to scripts, diagrams, or test results (where included)
- Learnings: Skills gained and insights
- Next steps: Possible improvements or follow-up experiments
